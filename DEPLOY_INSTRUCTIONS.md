# Инструкция по развертыванию проекта

## Шаг 1: Загрузка на GitHub

### Вариант A: Использование скрипта (рекомендуется)
1. Откройте PowerShell в директории проекта
2. Выполните команду:
   ```powershell
   .\deploy-to-github.ps1
   ```
3. Введите сообщение коммита или нажмите Enter для использования стандартного

### Вариант B: Ручное выполнение команд
Выполните следующие команды в PowerShell или Git Bash:

```bash
# Добавить все изменения
git add .

# Создать коммит
git commit -m "Update project files"

# Отправить на GitHub
git push origin main
```

## Шаг 2: Развертывание на Vercel

### Способ 1: Через веб-интерфейс Vercel (рекомендуется)

1. **Перейдите на сайт Vercel:**
   - Откройте https://vercel.com
   - Войдите в аккаунт (можно через GitHub)

2. **Импортируйте проект:**
   - Нажмите кнопку "Add New..." → "Project"
   - Выберите репозиторий `Slerz/fill_template`
   - Нажмите "Import"

3. **Настройте проект:**
   - Vercel автоматически определит настройки из `vercel.json`
   - Если нужно, добавьте переменные окружения (Environment Variables)
   - Нажмите "Deploy"

4. **Готово!**
   - После деплоя вы получите URL вашего сайта
   - Все последующие коммиты в `main` будут автоматически деплоиться

### Способ 2: Через Vercel CLI

1. **Установите Vercel CLI** (если еще не установлен):
   ```bash
   npm install -g vercel
   ```

2. **Войдите в Vercel:**
   ```bash
   vercel login
   ```

3. **Деплой проекта:**
   ```bash
   vercel
   ```
   
   При первом деплое:
   - Выберите "Link to existing project" или создайте новый
   - Следуйте инструкциям

4. **Продакшн деплой:**
   ```bash
   vercel --prod
   ```

## Структура проекта для Vercel

Проект уже настроен для Vercel через `vercel.json`:
- API функции в папке `api/` будут работать как serverless функции
- Статические файлы будут раздаваться автоматически

## Важные замечания

1. **Переменные окружения:**
   - Если ваш проект использует API ключи или другие секреты, добавьте их в настройках проекта на Vercel
   - Перейдите в Project Settings → Environment Variables

2. **Автоматический деплой:**
   - После первого деплоя все коммиты в ветку `main` будут автоматически деплоиться
   - Это можно настроить в Project Settings → Git

3. **Домены:**
   - Vercel предоставляет бесплатный домен вида `your-project.vercel.app`
   - Можно добавить свой домен в Project Settings → Domains
