# Chat Landing Page

Простой чат-лендинг с AI-консультантом.

## Установка

1. Установите зависимости:
```bash
npm install
```

## Запуск

Запустите локальный сервер:
```bash
npm start
```

Или с автоматическим открытием браузера:
```bash
npm run dev
```

Сайт будет доступен по адресу: http://localhost:8080

## Структура проекта

- `index.html` - главная страница
- `styles.css` - стили
- `chat.js` - логика чата
- `prompts/` - папка с промптами для бота
  - `sys-prompt.txt` - системный промпт
  - `user-prompt` - база знаний
